### Archived Projects

Individual course projects are preserved in the `projects/` directory:

- **[Project 1: Personal Portfolio](projects/project1-original/)** - Original portfolio submission featuring HTML and CSS fundamentals
- **[Project 2: Reading Log Tracker](projects/project2-tracker/)** - Interactive reading log with search, filter, sort, and note-taking capabilities using JavaScript, DOM manipulation, and localStorage
- **[Project 3: Personal Data Dashboard](projects/project3-dashboard/)** - Dynamic dashboard displaying weather data, inspirational quotes, tasks, and notes with theme toggle and real-time updates

## Key Features

### Main Portfolio Site
- **Responsive Design** - Mobile-first approach with breakpoints for tablet and desktop
- **Modern CSS** - Custom properties, gradients, glassmorphism effects, and smooth animations
- **Dark Mode Support** - Automatic theme switching based on system preferences
- **Accessibility** - Semantic HTML, ARIA labels, keyboard navigation, and reduced-motion support
- **Interactive Widgets** - Project showcase with hover effects and smooth transitions
- **Custom Typography** - Playfair Display for headings, Inter for body text

### Project 2: Reading Log Tracker
- Add, edit, and delete book entries
- Search by title or author
- Filter by reading status (Currently Reading, Completed, Want to Read)
- Sort by title, author, or date added
- Personal notes for each book
- Data persistence with localStorage

### Project 3: Personal Data Dashboard
- Weather information display
- Rotating inspirational quotes
- Task management with add/complete/delete
- Note-taking functionality
- Day/night theme toggle
- Smooth animations and transitions
- JSON data integration

### Project 4: Writing Tool
- AI-powered brainstorming and ideation
- A journal of writing details
- Assitance with story mapping
- Interactive writing interface
- Real-time feedback
